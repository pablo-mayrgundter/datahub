# Download It #

See the [Source](http://code.google.com/p/datahub/source/checkout) page for how to checkout the code.

# Build It #

```
cd <the code directory>
export GAE_HOME=<path to your appengine-java-sdk>
ant
```

# Run It #

```
ant server
```

You should now be able to view a web interface at http://localhost:8080/#/data.

You can also use the REST API on the homepage.
